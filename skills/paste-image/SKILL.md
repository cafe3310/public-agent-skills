# 技能: paste-image

## 描述

此技能提供一个脚本，用于将 macOS 剪贴板中的图片直接粘贴到指定的 PNG 文件。这是一种安全且可靠的方式，可以将用户的剪贴板中的图片数据获取到工作区。

## 何时使用

当需要从 macOS 系统的用户剪贴板中读取图片数据时，应使用此技能。例如，如果用户说“我的剪贴板里有一张图片，你能分析它吗？”。

## 命令

要使用此技能，请执行 `run.sh` 脚本，并将所需的输出路径作为参数。

`paste-image/scripts/run.sh <输出路径>`

### 参数

-   `<输出路径>` (必填): PNG 图片文件应保存的完整绝对路径。该路径的目录必须已经存在。

## 错误处理

-   如果未提供输出路径，脚本将报错退出。
-   如果输出路径的目录不存在，脚本将报错退出。
-   如果剪贴板中不包含有效的 PNG 格式图片，脚本将报错退出。在这种情况下，任何可能已创建的空文件都将被自动删除。
